# Dev Collaboration Platform

## Overview
A full-stack SaaS-style developer collaboration platform built with React, Express, PostgreSQL, JWT authentication, and Socket.IO.

The platform allows users to create projects, manage team membership, and communicate through real-time project chat.

This project demonstrates full-stack architecture, relational database design, authentication systems, and realtime WebSocket communication.

---

## Features

### Authentication
- Register new user
- Reject duplicate email
- Login valid user
- Reject invalid login
- Logout

### Dashboard
- View user projects
- Create new project
- Delete project

Project page
- Open project
- Load members
- Load messages
- Send message
- Refresh page
